Projet 2 : Mini Application Web

Description du projet

Ce projet est une petite application web permettant de gérer des données simples. Elle a été conçue pour apprendre les bases du DOM, des événements et de l’interaction utilisateur.

Fonctionnalités

  • Interface simple et intuitive
  • Gestion dynamique des éléments
  • Interactions JavaScript en temps réel
  • Design responsive et épuré
